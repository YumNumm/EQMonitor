## **機能**
- [x] リアルタイム震度の表示
- [x] 緊急地震速報・地震情報の通知
- [x] 通知の自動ツイート機能
- [x] 地震履歴の表示
- [ ] PGA(リアルタイム加速度)の表示
- [ ] P,S波到達予想円の表示
- [ ] 震源地からの距離の表示
- [ ] 緊急地震速報の表示(通知ではない)
- [ ] 震度の色分け
- [ ] 震源地の表示
